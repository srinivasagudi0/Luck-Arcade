# Luck Arcade

Modes:
- CLI mode (`main.py`)
- Flask mode (`app.py`)

Games:
- Dice Roll
- Coin Flip
- Rock Paper Scissors
- Meteor Dodge
- Planet Guess
- Number Guess

Stats persist in `stats.json`.

## Run the UI

```bash
pip install -r requirements.txt
python3 app.py
```

Open `http://127.0.0.1:8000`.

## Run the CLI

```bash
python3 main.py
```
